域用于资源的分类，在租户视图下，可以对VDC中的虚拟机进行分域。

通过给域中添加不同的用户，为用户配置不同的权限，使得不同的用户可以对不同的虚拟机拥有不同的权限。

创建域

当系统中默认的域无法满足您的权限需求时，您可以通过添加域来创建一个新的域。

在“用户 > 分域管理”页面，单击“添加”。

删除域

当某个域不需要再使用时，可以将其删除。

在“用户 > 分域管理”页面，选择需要删除的域，单击“删除”。

添加域用户

如果需要某个用户在某个域中具有权限，可以通过为域中增加成员的方式实现。

在“用户 > 分域管理”页面，在右侧的“详细信息”区域框，在“用户”页签单击“添加成员”。

移除用户

如果某个用户不需要在某个域中更具有权限，可通过将此用户从域中移除来实现。

在“用户 > 分域管理”页面，在右侧的“详细信息”区域框，在“用户”页签，选择需要移除的用户，单击“移除用户”。

配置权限

如果某个用户的权限在域中需要变更时，可以通过为用户配置权限来实现。

在“用户 > 分域管理”页面，在右侧的“详细信息”区域框，在“用户”页签，选择需要配置权限的用户，单击“修改权限”。

添加虚拟机

需要对某个虚拟机进行分域管理时，可以通过为域中添加虚拟机来实现。

在“用户 > 分域管理”页面，在右侧的“详细信息”区域框，在“虚拟机”页签，单击“批量添加”，选择需要添加至域中的虚拟机。

删除虚拟机

某个已经被添加至域中的虚拟机不需要再进行分域管理时，可以通过将虚拟机从域中删除来实现。

在“用户 > 分域管理”页面，在右侧的“详细信息”区域框，在“虚拟机”页签，选择需要从域中移除的虚拟机，单击“删除”。
