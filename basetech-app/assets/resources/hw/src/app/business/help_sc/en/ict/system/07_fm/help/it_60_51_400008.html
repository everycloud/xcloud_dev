Access Control List (ACL) is an access security control method that allows permitted users to access only the specified network resources by controlling the data packets sent to or from a port.

The ACL function is available to use only after the VPC interconnection function is enabled in the system.

Enabling an ACL

Enable an ACL before adding rules for it. Before enabling an ACL, you need to apply for a router for the VPC in which the ACL rules takes effect.

Choose VPC > My VPC, select a VPC, choose Security > ACLs, and click Enable ACL.

Disabling an ACL

Disable an ACL as needed. If an ACL is disabled, rules cannot be added to the ACL. If a VPC already has ACL rules added, the ACL can be disabled for the VPC only after all the ACL rules have been deleted from the VPC.

Choose VPC > My VPC, select a VPC, choose Security > ACLs, and click Disable ACL.

Adding an ACL Rule

Add an ACL rule to a network in a VPC based on the service requirements.

ACL rules are sorted by priority. The newest-added ACL rule appears at the end of the ACL list with the lowest priority. To add an ACL rule with its priority higher or lower than that of a target ACL rule, locate the row that contains the target ACL rule and click Add a Rule Above or Add a Rule Below to add the rule as needed.

Choose VPC > My VPC, select a VPC, choose Security > ACLs, and click Add Rule.

Deleting an ACL Rule

If an ACL rule is not needed any more, you can delete it from the system.

Choose VPC > My VPC, select a VPC, choose Security > ACLs, and click Delete.

Adding a Rule Above

In the list of existing ACL rules, add an ACL with its priority higher than that of a target ACL rule.

Choose VPC > My VPC, select a VPC, choose Security > ACLs, locate the row that contains the target ACL rule, click More, and select Add a Rule Above.

Adding a Rule Below

In the list of existing ACL rules, add an ACL with its priority lower than that of a target ACL rule.

Choose VPC > My VPC, select a VPC, choose Security > ACLs, locate the row that contains the target ACL rule, click More, and select Add a Rule Below.
