Browser Configuration

Scenarios

  • Configure the browser before you log in to ServiceCenter for the first time. If you have configured the browser when installing it, skip this procedure.
  • Both Internet Explorer and Mozilla Firefox need to be configured.

Prerequisites

  • You have installed the browser of the required version.
  • You have obtained the floating IP address of the ServiceCenter node.

Internet Explorer Configuration

The following uses Internet Explorer 9.0 as an example to describe how to configure Internet Explorer.

    Set the compatibility view.

    1. Open Internet Explorer.
    2. Enter the network address of ServiceCenter in the address box and press Enter.

      The address is in http://Floating IP address of ServiceCenter format.

      For example, enter http://192.168.40.2.

    3. Click Continue to this website (not recommended).

      The ServiceCenter login page is displayed.

    4. Check whether an information dialog box is displayed indicating that the browser version does not meet the requirement.

      • If yes, go to 5.
      • If no, go to 12.

    5. Choose Tools > Compatibility View Settings.

      The Compatibility View Settings dialog box is displayed.

    6. Deselect Include updated website lists from Microsoft, Display intranet sites in Compatibility View, and Display all websites in Compatibility View, and click Close.
    7. Close all open pages on Internet Explorer.

    Set the developer tools.

    1. Open Internet Explorer again and repeat 2 to 3.
    2. Check whether an information dialog box is displayed indicating that the browser version does not meet the requirement.

      • If yes, go to 10.
      • If no, go to 12.

    3. Press F12 and set Browser to Internet Explorer 9 in the menu bar of the Developer Tools window.

      Document Mode is automatically set to Internet Explorer 9 Standards(Page Default).

    4. Close the Developer Tools window.

    Set Internet options.

    1. In Internet Explorer, choose Tools > Internet Options.

      The Internet Options dialog box is displayed.

    2. Click the Advanced tab.
    3. Deselect Warn about certificate address mismatch and click Apply.
    4. Click the Security tab.
    5. Click Custom level on the lower-right corner of the page.

      The Security Setting-Internet Zone dialog box is displayed.

    6. In the Settings area, set Automatic prompting for file downloads for Download to Enable.
    7. Click OK.

      The Warning dialog box is displayed.

    8. Click Yes.

      The Internet Options dialog box is displayed.

    9. Click OK.

      The settings take effect after the browser is restarted.

    Set security parameters.

    1. Check whether the operating system (OS) in use is Windows Server 2008.

      • If yes, go to 22.
      • If no, no further action is required.

    2. In the OS, click in the task bar.

      The Server Manager page is displayed.

    3. Choose Server Summary > Security Information and select Configure IE ESC.

      The Internet Explorer Enhanced Security dialog box is displayed.

    4. Set both Administrators and Users to Off and click OK.

      The enhanced security settings of Internet Explorer are complete.

Mozilla Firefox Configuration

The following uses Mozilla Firefox 24.0 as an example to describe how to configure Mozilla Firefox.
  1. Open Mozilla Firefox.
  2. Choose Tools > Options.

    The Options dialog box is displayed.

  3. Choose Advanced > Update.
  4. Set the update policy to Check for updates, but let me choose whether to install them.

    NOTE:
    Do not update Mozilla Firefox to a version other than Mozilla Firefox 21.0 to Mozilla Firefox 30.0 that are compatible with ServiceCenter.

  5. Click OK.
  6. Enter the network address of ServiceCenter in the address box and press Enter.

    The address is in http://ServiceCenterfloating IP address format.

    For example, enter http://192.168.40.2.

  7. Expand I Understand the Risks and click Add Exception.

    The Add Security Exception window is displayed.

  8. Select Permanently store this exception and click Confirm Security Exception.

    Mozilla Firefox is properly configured.
