Scenarios

On ServiceCenter, mount a CD/DVD-ROM drive of the local PC or an ISO file to a VM to facilitate operating system (OS) and application installation on the VM. Table 1 describes the mounting methods. You can select a proper method as required.
Table 1 Methods of mounting a CD/DVD-ROM drive or an ISO file

Mounting Method

Advantage

Limitation

Local mode
  • Supports ISO files and physical CD/DVD-ROM drives.
  • Simplifies operations.

Applies only to the following OSs:

  • SUSE Linux Enterprise Server 10 SP, SUSE Linux Enterprise Server 11 SP and later
  • Ubuntu 11.10 Server and later
  • Ubuntu 12.04 Desktop and later
  • Ubuntu 12.04 Server and later
  • Ubuntu 12.10 Server
  • Red Hat 3.0 32-bit
  • Red Hat 3.4 32-bit
  • CentOS 6.1 and later
  • Red Hat 6.1 and later
  • Oracle Linux Server release 6.1
  • Debian GUN/Linux 6.0.4 and later
  • openSUSE 11.3 64-bit
  • Red Flag Asianux Server 4 SP2
  • Windows of all versions

Sharing mode

Has no limitation on the OS.
  • Supports only ISO files.
  • Requires the setting of ISO file sharing parameters.

Precautions

  • Only one CD/DVD-ROM drive or ISO file can be mounted to a VM.
  • You can mount CD/DVD-ROM drives or ISO files only for a maximum of four VMs on a host. Therefore, if the CD/DVD-ROM drive or ISO file is not used, unmount it. Otherwise, mounting a CD/DVD-ROM drive to another VM running on the host fails.
  • A VM cannot be migrated when a CD/DVD-ROM drive or an ISO file is being mounted to the VM.
  • The CD/DVD-ROM drive or ISO file will be automatically unmounted from the VM after the VM is stopped, hibernated, migrated, or restarted using HA. If the CD/DVD-ROM drive or ISO file is mounted in local mode to a VM, it will be automatically unmounted from the VM after the VNC window is closed.

Prerequisites

  • The PC used for logging in to ServiceCenter can communicate with the ServiceCenter management network. If they are isolated from each other, a virtual private network (VPN) software is configured on the PC to allow the PC user to access the ServiceCenter management network. For details about VPN configuration, see Installing and Configuring VPN (Windows).
  • You have logged in to ServiceCenter.
  • The VM is in the Running state.

Mounting a CD/DVD-ROM Drive or an ISO File in Local Mode

  1. Log in to the VM using Virtual Network Computing (VNC) and mount a CD/DVD-ROM drive or an ISO file to the VM.

    During the mounting process, do not close the VNC window. Otherwise, the CD/DVD-ROM drive or ISO file is automatically unmounted.

  2. Click and select Mount Local CD/DVD-ROM Drive, as shown in Figure 1.

    Figure 1 Mounting a CD/DVD-ROM drive or an ISO file in local mode

  3. Select the directory that contains the CD/DVD-ROM drive or ISO file, as shown in Figure 2.

    • If you want to mount a CD/DVD-ROM drive, select CD/DVD-ROM and the directory that contains the CD/DVD-ROM drive.
    • If you want to mount an ISO file, select File, click Browse, and select the ISO file.
    Figure 2 Mounting a CD/DVD-ROM drive or an ISO file in local mode

  4. To mount an OS installation file and install the OS for the VM, select Restart the VM now to install the OS.
  5. Click OK.

    If Restart the VM now to install the OS is already selected, click Yes in the Connection dialog box that is displayed to reconnect the VM. Otherwise, the CD/DVD-ROM drive or ISO file is automatically unmounted.

Mounting an ISO File in Sharing Mode

  1. Log in to the VM using VNC and mount an ISO file to the VM.

    During the mounting process, do not close the VNC window. Otherwise, the ISO file is automatically unmounted.

    NOTE:

    When creating a VM template, you can set sharing parameters on the ServiceCenter page.

  2. Click and select Mount Shared CD/DVD-ROM Drive, as shown in Figure 3.

    Figure 3 Mounting an ISO file in sharing mode

  3. Set ISO file sharing parameters, as shown in Figure 4.

    Figure 4 Mounting an ISO file in sharing mode

  4. To mount an OS installation file and install the OS for the VM, select Restart the VM now to install the OS.
  5. Click OK.

    If Restart the VM now to install the OS is already selected, click Yes in the Connection dialog box that is displayed to reconnect the VM. Otherwise, the CD/DVD-ROM drive or ISO file is automatically unmounted.
