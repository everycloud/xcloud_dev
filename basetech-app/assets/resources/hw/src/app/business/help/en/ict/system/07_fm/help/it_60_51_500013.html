FusionManager provides performance monitoring indicators of the resources in the FusionSphere OpenStack connected to it.

To view performance indicators, choose Monitoring > Monitoring View.

CPU Usage

Definition

Measures the CPU usage of a measurement object.

Formula

VM CPU usage = CPU usage of a VM/Number of VM CPU cores

Description

None

Memory Usage

Definition

Measures the memory usage of a measurement object.

Formula

VM memory usage = Memory usage of a VM/Total memory of the VM

Description

None

Network Inbound

Definition

Measures the amount of inbound network traffic per second of a measurement object.

Formula

Network inbound rate of a VM = Number of inbound bytes of the VM/Measurement period

Description

None

Network Outbound

Definition

Measures the amount of outbound network traffic per second of a measurement object.

Formula

Network outbound rate of a VM = Number of outbound bytes of the VM/Measurement period

Description

None

Disk Write Speed

Definition

Measures the amount of data written to the measurement object per second.

Formula

Disk writes of a VM = Total number of bytes written to VM disks/Measurement period

Description

None

Disk Read Speed

Definition

Measures the amount of data read from the measurement object per second.

Formula

Disk reads of a VM = Total number of bytes read from VM disks/Measurement period

Description

None

Disk Usage

Definition

Measures the disk usage of a measurement object.

Formula

VM Disk usage = Occupied disk capacity of a VM/Total disk capacity of a VM

Description

None
