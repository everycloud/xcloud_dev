Public IP Addresses

  • Applying for a Public IP Address
  • Releasing a Public IP Address

A public IP address allows a user to access a VM from a public network. You can apply for a public IP address in a virtual private cloud (VPC) to support public network services, such as elastic IP addresses, destination network address translation (DNAT), and source network address translation (SNAT).

Applying for a Public IP Address

Before using a public IP address, ensure that you have applied for the public IP address from the FusionManager public IP address pool for the VPC.

Choose VPC > My VPC, select a VPC, choose Network > Public IP Address, and click Apply For.

Releasing a Public IP Address

If an obtained public IP address is not in-use, you can release the public IP address to save public IP address resources.

Choose VPC > My VPC, select a VPC, choose Network > Public IP Address, and click Release.
