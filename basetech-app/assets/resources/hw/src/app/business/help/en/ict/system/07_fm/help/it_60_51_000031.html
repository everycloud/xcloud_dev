Overview

A virtual machine (VM) is a software computer that functions like a physical computer, running an operating system (OS) and applications.

Provisioning a VM is a process of creating a VM with specified configurations.

Provisioning Operation

Choose Resources > VMs and use an existing VM template and VM specifications set to create the required VM.
