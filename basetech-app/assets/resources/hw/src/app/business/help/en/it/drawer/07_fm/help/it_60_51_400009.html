Security Groups

Security group is a VM access security control method that specifies the communication scope of VMs in the same or different security groups. You can define different access rules for a security group, and these rules take effect for all VMs added to this security group.

Creating a Security Group

Create security groups in a virtual private cloud (VPC) to control the permission to access VMs in the VPC.

Modifying a Security Group

Modify the name and description of a security group if needed.

Adding a Rule for a Security Group

Add a rule for VMs not included in a security group to communicate with VMs within the security group.

Deleting a Security Group Rule

If a security group rule is not needed any more, you can delete it from the system.

Deleting a Security Group

If a security group is not needed any more, you can delete it from the system. However, if the security group contains security group members, it cannot be deleted.
