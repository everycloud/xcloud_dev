My VPC

The My VPC page displays the information about the custom virtual private clouds (VPCs) created by the administrator.

A VPC provides a secure, isolated network environment in a virtual data center (VDC). You can customize virtual networks that provide the same functions as traditional networks in a VPC. The custom virtual networks can also provide advanced network features, such as elastic IP address and security group, to keep up with service deployment requirements.

Creating a VPC

FusionManager provides multiple VPCs that have networks configured by default after being created. You can also create more networks in these VPCs. When creating such VPCs, if a network fails to be created due to insufficient resources or incorrect configurations, the failure does not affect the creation of other networks in the VPC or the creation of the entire VPC.

You can also customize an empty VPC and create desired network resources in the VPC.

A VPC is equivalent to a project in FusionSphere OpenStack.

Modifying Resource Quotas

You can modify the quotas of resources available for a VPC as needed.

Modifying VPC Information

You can modify the name and description of a VPC.

Deleting a VPC

If a VPC is not needed any more, you can delete it from the system.
