Service Provisioning Overview

Overview

You can provision the following services using the resources available in a virtual data center (VDC).
  • VMs

    A virtual machine (VM) is a software computer that functions like a physical computer, running an operating system (OS) and applications.

  • Application instances

    An application instance is a collection of VMs and service software that provides specific services, for example, SQL Server, customer relationship management (CRM), and enterprise resource planning (ERP).

A virtual private cloud (VPC) provides tenants with isolated and secure networks. You can create virtual networks in a VPC to provide tenants with the same network functions as those provided by a traditional network. Those virtual networks can also run VMs and application instances.
