Summary

On the Summary page, you can view information about network resource usages in a virtual private cloud (VPC).

Network Topology

A network topology shows the direct networks, internal networks, and routed networks created in a VPC, and also shows the VMs associated with all networks.

Resource Quotas

In the Resource Quotas area, you can view the resource quotas allowed for a VPC and the resource usages in the VPC.

Resource Statistics

The Network Statistics chart shows the numbers of various networks in the VPC. The Elastic IP Address chart shows the numbers of elastic IP addresses used by different services in the VPC and the number of available elastic IP addresses in the VPC.
