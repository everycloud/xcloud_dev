Provisioning an Application Instance

Description

An application instance is a collection of VMs and service software that provides specific services, for example, SQL Server, customer relationship management (CRM), and enterprise resource planning (ERP).

Provisioning an application instance is a process of creating an application instance.

Provisioning Method

You can provision application instances using a method shown in Figure 1 based on the mapping scenario listed in Table 1. Table 2 lists the involved tasks.

Figure 1 Provisioning method

Table 1 Scenario

Provisioning Mode

Scenario

1

Create an application template based on the service requirements and use the template to provision an application instance. This mode applies to a newly-deployed FusionManager system.

2

Associate multiple VMs to form an application instance. This mode applies to scenarios in which the VMs carrying the required services already exist in the system, or in which the required services only need a small number of VMs. Typically, an application template is only used to create a large number of VMs in batches.
