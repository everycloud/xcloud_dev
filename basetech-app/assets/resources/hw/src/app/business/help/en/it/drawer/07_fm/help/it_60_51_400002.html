Routers

You can apply for a virtual router in a virtual private cloud (VPC) to create routed networks. Routed networks in a VPC can communicate with each other. VMs in a routed networks can use advanced network functions including elastic IP address, destination network address translation (DNAT), source network address translation (SNAT), access control list (ACL), and virtual private network (VPN). FusionManager supports the following types of routers:
  • Hardware router: a firewall instance created on a physical firewall to provide router functions
  • Software router: a system service VM, which is also referred to as a virtual firewall (VFW), created on FusionManager to provide router functions
Hardware routers and software routers support the following advanced network functions:
  • Hardware router: elastic IP address, NAT, ACL, ASPF and VPN
  • Software router: elastic IP address, NAT, ACL, VPN (IPSec type), and virtual extensible LAN (VXLAN)

Applying for a Router

A VPC requires only one virtual router.

  • Before applying for a hardware router, you need to add a physical firewall to FusionManager and configure the physical firewall as instructed.
  • Before applying for a software router, you need to create an external network and Virtual Service Appliance (VSA) management network, and import a VSA VM template to the system. The created external network must meet the following requirements: its connection mode is subnet (common VLAN), it is connected to the Internet, its subnet IP address assignment mode is static injection, and it does not contain an IPv6 subnet. If the software router is also required to support the VXLAN functions, you need to create a VXLAN Tunnel Endpoint (VTEP) network.

Releasing a Router

If a router is not needed any more, you can delete it from the system to release router resources. However, if the router is already associated with a network, the router cannot be released.

Network Detection

The network detection function is used to detect the connectivity between a router and a network address.
