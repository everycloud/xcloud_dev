DNAT

The DNAT page displays all destination network address translation (DNAT) services obtained in the current cloud resource pool.

Destination network address translation (DNAT) enables FusionManager to map an internal network address to a public network address. If an external address initiates a request to set up a service connection with FusionManager, the gateway configured on the system firewall receives the connection request and translates the external address to an internal address, so that the external address can communicate with the system and use external services provided by the system.
