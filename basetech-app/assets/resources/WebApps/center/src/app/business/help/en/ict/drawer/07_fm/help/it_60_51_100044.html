VM-Summary

The general information about the VRM VM is displayed, including basic properties, number of alarms, resource usage, and NIC and disk information.

VM Power Management

Power management for a VM includes hibernate, start or wake up, stop, restart, pause, unpause the VM. Stop or restart a VM can be performed in a safe or forcible manner. You are advised to use the safe manner. Forcibly stop or restart a VM only if the VM cannot be safely stopped or restarted due to an OS fault. Forcibly stopping or restarting a VM may damage the VM disks and cause data loss.

Starting or stopping a large number of VMs increases host workload. If you need to start or stop a large number of VMs on a host, perform the start or stop operations in batches to prevent VM services on the host from being adversely affected.

Migrating a VM

For details about requirements, see VM Migration and Specification Modification Requirements.

Live migration

If the service load is over light or heavy on the host where the VMs run or the host is faulty, migrate the VMs to ensure that they work properly.
  • If the service load is too heavy on the host or the host is faulty, migrate the VMs to another host to prevent service interruption due to a host exception.
  • If the service load is too light on multiple hosts, migrate the VMs to a small number of hosts to consolidate host resources and then power off the idle hosts. This improves resource utilization and reduces power consumption.

Click More and select Live Migrate.

Cold Migration

Click More and select Cold Migrate.

After cold migration, you also need to click More and select Confirm Modification to make the system continue the subsequent operations.

FAQ

Why is the VM performance under expectation and the VM lacking part of monitoring data when UVP VMTools is not installed?

UVP VMTools is a recommended component for VMs. Without UVP VMTools, the information monitoring and input/output (I/O) performance of a VM are adversely affected. However, UVP VMTools only applies to Huawei FusionSphere OpenStack hypervisor. For details about installing UVP VMTools on a VM, see FusionSphere document FusionSphere OpenStack Image Creation Guide.
