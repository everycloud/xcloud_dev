FusionStorage

  • Adding FusionStorage to FusionManager
  • Deleting a FusionStorage

FusionStorage is Huawei's proprietary distributed storage software that can be deployed on servers to integrate computing and storage resources. If deployed on a general x86 server, FusionStorage virtualizes all the local hard disks on the server into a virtual storage pool to provide block storage functions.

FusionStorage needs to be added to FusionManager only when FusionStorage is required in the cloud system. After being added to FusionManager, FusionManager can monitor the running conditions of FusionStorage and receive alarms reported from FusionStorage.

For more information about FusionStorage, see the FusionStorage documentation, which can be obtained at Documentation.

Adding FusionStorage to FusionManager

Choose Resource > Devices > Storage Devices > FusionStorage and click Add FusionStorage.

The user account used to add FusionStorage to FusionManager is gmadmin, and its default password is gmHuawei@123.

After FusionStorage is successfully added to FusionManager, the FusionStorage information is displayed on the FusionStorage page.

Deleting a FusionStorage

Delete the FusionStorage that is no longer used. After the FusionStorage is deleted, it is no longer managed and monitored by the FusionManager system but its resources can be still used.

Choose Resource > Devices > Storage Devices > FusionStorage, locate the row that contains the target FusionStorage, and click Delete.
