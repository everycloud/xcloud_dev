监控视图

提供接入FusionManager的资源的集群、主机、虚拟机的性能监控指标。下文以FusionCompute为例介绍各项性能指标的具体含义。

CPU使用率

指标含义

该指标用于统计测量对象的CPU使用率。

测量对象可以是：集群、主机、虚拟机。

计算公式

  • 集群CPU使用率 = 集群中各主机CPU使用率之和 / 主机数量。
  • 主机CPU使用率 = （主机dom0的CPU使用率*dom0的核数+主机domU的CPU使用率*domU的核数）/（dom0的核数+ domU的核数）。
  • 虚拟机CPU使用率 = 单个虚拟机CPU使用率 / 单个虚拟机的CPU总核数。

相关说明

无。

CPU预留率

指标含义

统计测量对象中，当前预留的CPU数量占CPU总量的百分比。

测量对象可以是：集群。

计算公式

CPU预留率 = CPU预留容量 / (CPU预留容量 + CPU可用容量) x 100%

其中
  • CPU预留容量：指测试对象下，所有运行态虚拟机预留CPU数量的总和。
  • CPU可用容量：指测试对象下，可分配给虚拟机使用的CPU数量的总和。
说明：

在VMware虚拟化环境中，CPU预留容量不包含VMware系统自身预留的容量。

相关说明

该指标的正常取值范围为“0%～100%”。
  • 若该指标等于100%，说明该测量对象下无CPU资源可分配。
  • 若该指标等于0%，说明该测量对象下无预留CPU资源，所有CPU资源均可分配使用。

内存使用率

指标含义

该指标用于统计测量对象的内存使用率。

测量对象可以是：集群、主机和虚拟机。

计算公式

  • 集群内存使用率 = 集群中各主机内存使用率之和 / 主机数量。
  • 主机内存使用率 = （domainU内存使用量+domain0内存使用量）/（domainU内存总量+domain0内存总量）。
  • 虚拟机内存使用率 = 该虚拟机内存使用量 / 该虚拟机内存总量。

相关说明

无。

内存预留率

指标含义

统计测量对象中，当前预留的内存数量占内存总量的百分比。

测量对象可以是：集群

计算公式

内存预留率 = 内存预留容量 / (内存预留容量 + 内存可用容量) x 100%

其中
  • 内存预留容量：
    • 内存复用关闭时，指测试对象下，所有运行虚拟机的内存与显存总和。
    • 内存复用开启时，指测试对象下，所有运行虚拟机的预留与显存总和。
  • 内存可用容量：指测试对象下，可分配给虚拟机使用的内存数量总和。
说明：

在VMware虚拟化环境中，内存预留容量不包含VMware系统自身预留的容量。

相关说明

该指标的正常取值范围为“0%～100%”。
  • 若该指标等于100%，说明该测量对象下无内存资源可分配。
  • 若该指标等于0%，说明该测量对象下无预留内存资源，所有内存均可以分配使用。

存储使用率

指标含义

统计当前已经使用的存储容量占存储总量的百分比。

测量对象可以是：集群。

计算公式

存储池使用率 = 已使用存储容量 / (已使用存储容量 + 剩余存储容量) x 100%

其中
  • 已使用存储容量：指测试对象下，已经使用的存储容量总和。
  • 剩余存储容量：指测试对象下，剩余未使用的存储容量总和。

相关说明

该指标的正常取值范围为“0%～100%”。
  • 若该指标等于100%，说明该测量对象下无存储空间可以分配。
  • 若该指标等于0%，说明该测量对象所有存储空间均未被使用。

存储分配率

指标含义

统计测量对象中，当前已经分配的存储数量占存储总量的百分比。

测量对象可以是：集群。

计算公式

存储池分配率 = 已分配存储容量 / (已使用存储容量 + 剩余存储容量) x 100%

其中
  • 已分配存储容量：指测试对象下，已经分配的存储容量总和。
  • 已使用存储容量：指测试对象下，已经实际使用的物理存储容量总和。
  • 剩余存储容量：指测试对象下，剩余未使用的物理存储容量总和。
说明：
例如，存储池实际物理存储空间为100GB，自动精简配置后，可用于分配的存储空间为200GB。当前为虚拟机A分配了50GB，虚拟机占用20GB用于文档存储；为虚拟机B分配了80GB，虚拟机占用10GB用于数据库存储。则：
  • 已分配存储容量：130GB
  • 已使用存储容量：30GB
  • 剩余存储容量：70GB
  • 存储池分配率：130%

相关说明

该指标的取值下限为“0%”，上限由存储自动精简配置比例决定。

网络流入速率

指标含义

该指标用于统计每秒流入测量对象的网络流量。

测量对象可以是：主机、虚拟机。

计算公式

  • 主机网络流入流速 = 主机的网络流入的字节数之和 / 测量周期。
  • 虚拟机网络流入流速 = 虚拟机的网络流入的字节数之和 / 测量周期。

相关说明

无。

磁盘使用率

指标含义

该指标用于统计测量对象的磁盘使用情况。

测量对象可以是：主机、虚拟机。

计算公式

  • 主机磁盘使用率 = 主机磁盘的已使用容量 / 主机磁盘的总容量。
  • 虚拟机磁盘使用率 = 虚拟机磁盘使用容量 / 虚拟机磁盘总容量。

相关说明

无。
