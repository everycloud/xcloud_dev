iNIC Driver Upgrade Process

During the system upgrade process, the system pushes the intelligent network interface card (iNIC) driver upgrade package to each host. Then each host pushes the package to each VM.

  • If the iNIC driver upgrade mode is set to Automatic, the system automatically upgrades the iNIC driver.
  • If the iNIC driver upgrade mode is set to Manual, the system displays a message to ask the VM user whether to upgrade the iNIC driver.

iNIC Driver Upgrade Modes

Both administrators and VM users can set the upgrade modes of Tools and intelligent network interface card (iNIC) drivers for VMs to fulfill user demands.

.
  • For an administrator: Set the PC driver upgrade mode for each VM on the FusionManager. For details, see Changing VM Upgrade Mode.
  • For a VM user: Click the Tools icon in the lower right corner on the VM desktop.

If the VM upgrade modes set by the administrator and the VM user are different, the VM upgrade mode in effect is Automatic.
