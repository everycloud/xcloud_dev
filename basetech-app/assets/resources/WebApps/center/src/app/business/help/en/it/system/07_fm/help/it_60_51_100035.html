VM Specifications

  • Modifying VM Specifications
  • Deleting VM Specifications

VM specifications define the VM hardware settings, including the number of CPUs, memory space, and system disk space.

You can define VM specifications for provisioning VMs with the defined specifications. VMs can have different specifications specified to meet different performance requirements.

Choose Resource > Templates and Specs > VM Specifications and click Create.

A maximum of 20 sets of VM specifications can be created.

Modifying VM Specifications

All the attributes of a VM can be modified as needed.

Choose Resource > Templates and Specs > VM Specifications, locate the row that contains the target VM specification set, and click Modify.

Deleting VM Specifications

If a set of VM specifications is not needed any more, you can delete it from the system.

Choose Resource > Templates and Specs > VM Specifications, locate the row that contains the target VM specification set, and click Delete.
