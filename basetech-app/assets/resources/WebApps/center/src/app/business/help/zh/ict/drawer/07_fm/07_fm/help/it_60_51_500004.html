监控视图

提供接入FusionManager的FusionSphere OpenStack的资源的性能监控指标。

CPU使用率

指标含义

该指标用于统计测量对象的CPU使用率。

测量对象可以是：主机、虚拟机。

计算公式

  • 主机CPU使用率 = 1–主机CPU空闲时间 / 主机CPU总时间。
  • 虚拟机CPU使用率 = 单个虚拟机CPU使用率 / 单个虚拟机的CPU总核数。

相关说明

无。

内存使用率

指标含义

该指标用于统计测量对象的内存使用率。

测量对象可以是：主机和虚拟机。

计算公式

  • 主机内存使用率 = 主机的内存使用量 / 主机的内存总量。
  • 虚拟机内存使用率 = 该虚拟机内存使用量 / 该虚拟机内存总量。

相关说明

无。

网络流入速率

指标含义

该指标用于统计每秒流入测量对象的网络流量。

测量对象可以是：主机、虚拟机。

计算公式

  • 主机网络流入流速 = 主机的网络流入的字节数之和 / 测量周期。
  • 虚拟机网络流入流速 = 虚拟机的网络流入的字节数之和 / 测量周期。

磁盘使用率

指标含义

该指标用于统计测量对象的磁盘使用情况。

测量对象可以是：主机、虚拟机。

计算公式

  • 主机磁盘使用率 = 主机磁盘的已使用容量 / 主机磁盘的总容量。
  • 虚拟机磁盘使用率 = 虚拟机磁盘使用容量 / 虚拟机磁盘总容量。
