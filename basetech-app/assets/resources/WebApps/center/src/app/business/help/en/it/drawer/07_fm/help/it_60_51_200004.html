Domain Management

A domain is used to classify resource clusters.

To allow different users to have different operation permissions for various resource clusters in a domain, you can add these users to the domain and assign different permissions to them.

Creating a Domain

If the default domain in the system cannot meet the requirements for fine-grained user rights management, you can create a new domain.

Choose User > Domain Management and click Add.

Deleting a Domain

If a domain is not needed any more, you can delete it from the system.

Choose User > Domain Management, select the target domain, and click Delete.

Adding a Domain User

To allow a user to have rights granted to a domain, you can add the user to the domain.

Choose User > Domain Management, navigate to the Users page in the Details area on the right pane, and click Add Member.

Removing a Domain Member

To disqualify a domain member from using rights granted to the domain, you can remove the user from the domain.

Choose User > Domain Management, navigate to the Users page in the Details area on the right pane, select the target user, and click Remove.

Modifying Rights of a User

You can modify the operation rights of a user if required.

Choose User > Domain Management, navigate to the Users page in the Details area on the right pane, select the target user, and click Modify Rights.

Adding a Resource Cluster to a Region

To have a specific resource cluster be managed by a region, you can add this resource cluster to the region.

Choose User > Domain Management, navigate to the Resource Cluster page in the Details area on the right pane, select the target resource cluster, and click Add Cluster.

Deleting a Resource Cluster from a Region

To have a specific resource cluster in a region out of management by the region, you can delete this resource cluster from the region.

Choose User > Domain Management, navigate to the Resource Cluster page in the Details area on the right pane, select the target resource cluster, and click Delete.
