Installing the Tools on a Windows VM

  • Scenarios
  • Prerequisites
  • Procedure
  • Additional Information

Scenarios

Tools provides drivers for VMs. It improves VM input/output (I/O) performance, monitors VM hardware, and enables other advanced functions on VMs. During VM template creation, install the Tools on the template.

For details about Tools functions, see Introduction.

Prerequisites

Conditions

  • The operating system (OS) has been installed on the VM.
  • Tools has not been installed on the VM.
  • The free space of the system disk is greater than 32 MB.
  • You have obtained the password of user Administrator if the VM runs the Windows Server 2008 or Windows 7 OS.
  • You have obtained the ISO file that contains the Tools installation file.

    The software package name is FusionCompute V100R005C00SPC200_GuestOSDrivers.zip, and the ISO file name is FusionCompute V100R005C00SPC200_GuestOS.iso.

    To obtain the software package
    • For enterprise users, visit http://support.huawei.com/enterprise and choose Software > IT > FusionCloud > FusionSphere > FusionCompute > V100R005C00SPC100.
    • For telecom carrier users, visit http://support.huawei.com and choose Software > Carrier Software & Core Network > Carrier Cloud Computing > IT Cloud Infrastructure Solution > Cloud OS > FusionCompute > V100R005C00SPC100.
    NOTE:
    Verify the obtained software packages. For details, see Verifying Software Packages.

Procedure

    Log in to the VM using VNC.

    1. Log in to the VM using Virtual Network Computing (VNC) and mount a CD/DVD-ROM drive or ISO file to the VM.

      The FusionCompute V100R005C00SPC200_GuestOS.iso is the required ISO file to be mounted.

      If the VM runs Windows Server 2008 or Windows 7, log in to the VM as user Administrator.

    Install the Tools.

    1. On the VM desktop displayed in the VNC window, choose Start > Computer.

      The Computer window is displayed.

    2. Right-click CD ROM, and choose Open.
    3. Right-click the Tools version based on the OS version in the pvDriver\windows folder, select Run as Administrator, and install the Tools as prompted.

      The Tools name contains the supported OS version and bit. For example, a Tools named pvdriver-win2008R2-64bit-x.x.x.x-x indicates that it supports Windows Server 2008 R2 64-bit.

    Follow-up Procedure

    After the Tools is installed, the VM NIC configuration information is lost. Therefore, if NIC information is configured before the installation, reconfigure the NIC.
