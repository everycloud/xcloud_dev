Hypervisor uniformly transforms storage units of storage resources into data stores that can provide hard disk resources for VMs.

After a cluster is successfully associated, the storage contained in the cluster is displayed on this page.

Exporting Storage Information

You can export storage information to your local PC.

Choose Resource > Resource Zones. Select the target resource zone, choose Computing Resource Pools, select the target resource cluster, and choose Storage. Then click Export.
