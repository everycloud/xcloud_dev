Changing a Password

Change your password after you log in to FusionManager for the first time and then periodically change the password to enhance account security.

To change the password, click to the right of the username in the upper-right corner of the page and select Change password.

Personal Settings

Personal settings are supplementary to your current information. You can set your mailbox as required so that:
  • The administrator can contact you by your mailbox.
  • In the alarm settings, your information will be automatically added to the default list.

To set your personal information, click username and choose Personal Settings in the upper right corner of the page.

NOTE:
When using required personal information, such as email addresses, ensure that the information use complies with local laws and regulations.

Email address is used for alarm notification by email. This function allows the system to send alarm information to the administrator or tenants using emails after an alarm is generated. The system administrator can view the email addresses specified for all VDC administrators and all tenants in the VDC. The administrator of a VDC can view the email addresses specified for all tenants in the VDC. After an email address is configured for a user, the email address will be kept in the system until being deleted from the system.

System Timeout

If you do not perform any operation within the specified time, you will be logged out automatically. A shorter system timeout duration provides higher security, but a longer timeout duration relieves the need to log in multiple times. You can set the system timeout duration based on your requirements.

After the system timeout duration is configured, the configuration does not take effect for the current user and takes effect the next time the user logs in to the system.

Choose System > System Timeout.
