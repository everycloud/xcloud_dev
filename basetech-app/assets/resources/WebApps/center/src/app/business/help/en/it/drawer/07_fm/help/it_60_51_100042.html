Tags

Tags are used to identify the classification or content of a target object.

Virtual resources with different performance levels can be attached with different tags to facilitate user's resource selection based on the service requirements.

You can define tags for identifying virtual resources, such as resources clusters and availability zones.

Modifying Tag Attributes

You can modify the attributes of a tag as needed.

Deleting a Tag or Tag Value

If a tag or tag value is not needed any more, you can delete it from the system. However, a tag or its tag value in use by a hypervisor cannot be deleted.

  • Deleting a tag: Locate the row that contains the target tag and click Delete.
  • Deleting a tag value: Locate the row that contains the target tag, click to expand the detailed tag information, and click Delete in the tag value column.
