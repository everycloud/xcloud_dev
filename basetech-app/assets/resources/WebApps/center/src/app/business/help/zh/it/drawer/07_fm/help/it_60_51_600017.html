配置DHCP服务器的网络信息

操作场景

使用FusionManager系统外部的DHCP服务器时，在交换机上配置DHCP服务器的网络信息，并为FusionManager上的网络配置DHCP中继。

前提条件

已使用终端仿真程序登录交换机设备。

操作步骤

    在交换机上配置DHCP服务器的网络信息

    1. 登录汇聚交换机。

      下列命令以华为交换机S5300为例进行说明，如果在其他型号交换机上执行该配置，具体命令请参考对应交换机的产品文档。

    2. 在系统视图中执行命令interface vlanif vlan id，进入DHCP服务器所在VLAN的vlanif视图。
    3. 执行命令ip address VLANIF的IP地址 子网掩码，配置VLANIF的IP地址和子网掩码。

      说明：

      如果DHCP服务器所在VLAN和其他网络为同一VLAN，且和其他网络使用不同的网关时，执行下面的命令配置从IP地址。

      ip address VLANIF的IP地址 子网掩码 sub

    4. 执行命令quit，退出VLANIF接口。

      例如，DHCP服务器所在VLAN ID为20，网关和子网掩码分别为“192.160.18.1”和“255.255.255.0”，在交换机上配置如下：

      <Quidway> system-view

      [Quidway] interface vlanif 20

      [Quidway-Vlanif20] ip address 192.160.18.1 24

      [Quidway-Vlanif20] quit

    为外部网络配置DHCP中继

    1. 登录汇聚交换机（以外部网络的网关在汇聚交换机上进行说明）。
    2. 在系统视图中执行命令dhcp enable，使能全局DHCP功能。
    3. 执行命令dhcp server group group-name，创建DHCP服务器组。
    4. 执行命令dhcp-server dhcp-server-ip-address，为DHCP服务器组添加DHCP服务器。
    5. 执行命令interface vlanif vlan id，进入外部网络对应的vlanif视图。
    6. 执行命令dhcp select relay，启用DHCP中继功能。
    7. 执行命令dhcp relay server-select DHCP服务器组，配置DHCP服务器IP地址。

      例如，外部网络的vlan id为2005，DHCP服务器的IP地址为192.160.18.10，在汇聚交换机上配置如下：

      <Quidway> system-view

      [Quidway] dhcp enable

      [Quidway] dhcp server group dhcpgroup1

      [Quidway-dhcp-server-group-dhcpgroup1] dhcp-server 192.160.18.10

      [Quidway-dhcp-server-group-dhcpgroup1] quit

      [Quidway] interface vlanif 2005

      [Quidway-Vlanif2005] dhcp select relay

      [Quidway-Vlanif2005] dhcp relay server-select dhcpgroup1
