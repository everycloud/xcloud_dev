Hosts

Hosts are the physical computers in an OpenStack instance running virtualization software and providing computing resources, such as CPUs and memory, for VMs.

After an OpenStack instance is successfully associated, the hosts contained in the OpenStack instance are displayed on this page.

Isolating a Host and Canceling Host Isolation

After a host is isolated, you cannot create or start VMs on it. If you need to replace parts on a host or perform maintenance for a host without stopping the host, isolate the host.
