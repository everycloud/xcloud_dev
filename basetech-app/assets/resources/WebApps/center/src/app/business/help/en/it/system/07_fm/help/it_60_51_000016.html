Overview

You can customize templates and specifications listed in Table 1 to facilitate service provisioning.
Table 1 Template and Specifications Categories

Category

Item

Description

Software resource templates

Software package

Software packages are installed on application VMs in an application template to deploy application instances. A software package contains software installation files, commands, and scripts.

Script

Scripts are included in application instance VMs to control software and process running in the VMs.

Service templates

VM template

A VM template is a duplicate of a VM, which includes an OS, software packages, and a set of VM specifications.

Application template

An application template is a service model that combines logical components, including VMs, software packages, scripts, and network resources, based on the configured relationship.

Resource specifications

VM specifications

VM specifications define the VM hardware settings, including the number of CPUs, memory space, and system disk space.

Virtual private cloud (VPC) specifications

Virtual private cloud (VPC) specifications are used to define the VPC configuration requirements, such as the number of allowed networks, and number of available elastic IP addresses, in a virtual data center (VDC).

IP bandwidth specifications

An IP bandwidth specification defines the maximum transmit and receive bandwidths for elastic IP addresses that can be used in a network.

Software Resource Configuration

Table 2 lists the software resource configuration tasks. You can perform the listed tasks as required.

Table 2 Software resource configuration

Resource

Configuration Task

Mandatory

Web Client Navigation Path

Software package
  • Automatic software installation: When you provision application instances using an application template in which the VM template has specific software configured, the system automatically installs the software on the VM that is to run the application instance.
  • Software distribution: After you select a software package and specify the target VMs to install the package, the system automatically installs the software on the specified VMs.

Optional

Choose Resource > Templates and Specs > Software Packages.

Script
  • Automatic script execution: When you provision application instances using an application template in which the VM template has specific script file configured, the system automatically executes the script on the VM that is to run the application instance.
  • Script distribution: After you select a script file and specify the target VMs to execute the script file, the system automatically distributes the script to the specified VMs and executes the script on them.

Optional

Resource > Templates and Specs > Scripts
