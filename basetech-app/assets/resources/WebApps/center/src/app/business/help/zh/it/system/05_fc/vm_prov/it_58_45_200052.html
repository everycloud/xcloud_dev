在Linux系统中安装智能网卡驱动

  • 操作场景
  • 必备事项
  • 操作步骤

操作场景

当主机使用智能网卡时，运行在该主机上的虚拟机必须安装智能网卡驱动，否则虚拟机的网卡不可见，导致虚拟机网络不通。

当有一台主机使用了智能网卡时，建议所有虚拟机都安装智能网卡驱动，避免虚拟机迁移至使用智能网卡的主机时，出现网络故障。

必备事项

前提条件

  • 已登录FusionManager。

  • 虚拟机的状态为“运行中”。

  • 虚拟机已安装Tools，且Tools运行正常。

  • 已获取智能网卡驱动所在的ISO文件。

    软件包名称为“FusionCompute V100R005C00SPC200_GuestOSDrivers.zip”，解压缩后的ISO文件名称为“FusionCompute V100R005C00SPC200_GuestOS.iso”。

    软件包下载路径：
    • 企业用户：“http://support.huawei.com/enterprise > 软件下载 > IT > 云计算 > 云操作系统 > FusionCompute > V100R005C00SPC100”
    • 运营商用户：“http://support.huawei.com > 产品软件 > 电信软件与核心网 > 电信云计算 > IT云基础设施解决方案 > Cloud OS > FusionCompute > FusionCompute V100R005 > FusionCompute V100R005C00SPC100”
    说明：
    获取到软件包后建议先执行软件包校验的操作，具体操作请参见校验软件包。

操作步骤

    判断是否需要安装智能网卡

    1. 在FusionManager界面，在“资源 > 设备 > 服务器”中选择主机。
    2. 在“硬件”页面，单击“网卡”。

      查看网卡的“型号”。

      根据型号辨识网卡是否为智能网卡。

    挂载光驱

    1. 使用“root”用户，通过VNC方式登录虚拟机，并挂载光驱。

      挂载光驱时需挂载“FusionCompute V100R005C00SPC200_GuestOS.iso”。

    2. 在VNC登录窗口的虚拟机操作系统界面，右键单击桌面，在弹出的快捷菜单中选择“在终端中打开”。

      进入命令行模式。如果linux操作系统无图形用户界面，直接在虚拟机操作系统界面执行命令操作。

    3. 执行以下命令，挂载虚拟机光驱。

      mount 挂载路径 xvdd

      例如：mount /dev/sr0 xvdd

    安装智能网卡驱动

    1. 执行以下命令，进入驱动包所在目录。

      cd xvdd/iNICDriver/linux

    2. 执行以下命令，查看智能网卡驱动。

      ls

      不同操作系统和智能网卡驱动的具体对应关系如表2所示。

      表2 操作系统和智能网卡驱动的对应关系

      操作系统

      驱动名称

      驱动所在路径

      DOPRA ICTOM V002R003 EIMP

      dopra_eimp_64_x.x.x.xx.tar.gz

      linux/inic-linux-customize

      DOPRA ICTOM V002R003 IMAOS

      dopra_imaos_64_x.x.x.xx.tar.gz

      Red Hat Enterprise Linux 3.0

      redhat3.0_32_x.x.x.xx.tar.gz

      Red Hat Enterprise Linux 3.4

      redhat3.4_32_x.x.x.xx.tar.gz

      其他操作系统

      inic-linux-x.x.x.xx.tar.gz

      linux

      以下操作以“inic-linux-x.x.x.xx.tar.gz”安装包为例，其余4种操作系统安装步骤类似。

    3. 执行以下命令，将所需的智能网卡驱动拷贝到root文件夹路径下。

      cp 驱动包名称 /root

      例如：cp inic-linux-x.x.x.xx.tar.gz /root

    4. 执行以下命令，解压缩智能网卡驱动包。

      cd /root

      tar -zxf 驱动包名称

      例如：tar -zxf inic-linux-x.x.x.xx.tar.gz

    5. 执行以下命令，安装智能网卡驱动。

      cd 驱动包所在目录

      ./install.sh

      例如：

      cd inic-linux

      ./install.sh

      说明：

      如果操作系统为Red Hat 3.0或Red Hat 3.4时，根据不同的内核版本，会有2个不同的驱动包目录，请根据内核版本选择安装目录和安装文件。

      执行如下命令可查看虚拟机操作系统的内核版本。

      uname -a

      当回显类似以下信息时，需选择含“smp”的安装目录和安装文件。
      Linux xxxx-SMP

    6. 执行以下命令，重启虚拟机。

      reboot

    验证安装结果

    1. 重启虚拟机后，使用“root”用户，通过VNC登录重新进入虚拟机操作系统界面。
    2. 在VNC登录窗口的虚拟机操作系统界面，右键单击桌面，在弹出的快捷菜单中选择“在终端中打开”。

      进入命令行模式。

    3. 执行以下命令，查看智能网卡的安装情况。

      rpm -qa|grep iNIC

      如果命令输出结果中包含网卡信息，说明驱动安装成功。回显示例如下：

      iNIC-SLES11.x-x.x.x.xx

    4. 卸载光驱。
