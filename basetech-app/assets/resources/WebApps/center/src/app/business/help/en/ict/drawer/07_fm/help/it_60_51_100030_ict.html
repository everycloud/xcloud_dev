VM Images

A VM image is a duplicate of a VM, which includes an OS, software packages, and a set of VM specifications.VM images can be used to:
  • Provision VMs. Creating VMs using a VM image can greatly reduce the time for creating the VMs and configuring operating systems (OSs) for them.
  • Provision application instances: A VM image can be used to create multiple VMs for an application instance. An application instance can be used to provision application instances.

Creating a VM Image

  1. Create an image file.

    Produce a VM image file based on FusionSphere OpenStack specifications.

    The image file must meet the following requirements:
    • The file format is qcow2, ami, aki, ari, iso, vhd, vmdk, raw, uec, vdi, vhdx, ovf, or img.
    • The file size is less than 6 GB.
    • The file has the UVP VMTools program installed. Otherwise, the VM information monitoring and input/output (I/O) performance may be adversely affected. However, UVP VMTools only applies to Huawei FusionSphere OpenStack hypervisor.
    For details about creating an image file, see FusionSphere OpenStack Image Creation Guide delivered for the FusionSphere solution.

  2. Register the VM image.

    Use the produced image file to register a VM image on the FusionManager web client.

    Choose Resource > Images and click Register.

Export the image.

Export the image file from FusionManager for creating VM images in other cloud environments.

  1. Locate the row that contains the target image, click More, and select Synchronize to synchronize the image file to FusionManager. If an image is not downloaded within one day after being synchronized to FusionManager, the image will be automatically deleted to release storage space. Therefore, download the synchronized images at the earliest time.

  2. Locate the row that contains the target image, click More, and select Download to download the image file to the local PC. After the image file is download, the image stored on FusionManager is automatically deleted to release storage space.

  3. After the image file is downloaded, locate the row that contains the image, click More, and select Clear to restore the image status so that the image can be synchronized and downloaded.
