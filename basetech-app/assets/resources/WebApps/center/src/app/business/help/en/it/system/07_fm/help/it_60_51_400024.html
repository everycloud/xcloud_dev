The VPN Links page displays all the virtual private network (VPN) connections created in the current cloud resource pool.

Virtual private network (VPN) connections are used to connect routed networks in a virtual private cloud (VPC) to user networks using VPN tunnels, which allow users to access service resources in the VPC.

FusionManager supports two types of VPNs: IP Security Protocol (IPSec) and Layer 2 Tunneling Protocol (L2TP).
  • IPSec VPN: commonly used for communication between enterprise headquarters and branches by setting up an encrypted VPN tunnel between the VPN gateway and the remote user network gateway.
  • L2TP VPN: commonly used for enterprise employees on business trips to connect to the enterprise network over the Internet. Such connections are set up between the VPN clients and the VPN gateway.
NOTE:
To use the L2TP VPN function in a shared VPC, the firewall must be Eudemon1000E V300R001C00.

Create a VPN Connection

Create an L2TP VPN connector for a shared VPC.

Choose VDC > VPC > VPN > VPN Connections and click Create VPN Connection.

Deleting a VPN Connection

If a VPN connection in a shared VPC is not needed any more, you can delete it from the system.

Choose VDC > VPC > VPN > VPN Connections, locate the row that contains the target VPN connection, and click Delete.

Modifying a VPN Connection Information

Modify information about a VPN connection in a shared VPC.

Choose VDC > VPC > VPN > VPN Connections, locate the row that contains the target VPN connection, and click Modify.
