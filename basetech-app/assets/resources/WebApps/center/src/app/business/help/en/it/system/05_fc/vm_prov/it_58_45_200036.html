Upgrading the Tools

  • Introduction
  • Set the Tools upgrade mode.

Introduction

During the system upgrade process, the system pushes the Tools upgrade package to each host. Then each host pushes the package to All VMs running on it.

NOTE:

Declaration: This feature is a high-risk feature. Using this feature complies with industry practices. However, end user data may be required for implementing the feature. Exercise caution and obtain end user's consent when using this feature.

The system pushes Tools to VMs during the upgrade process.

  • If the Tools upgrade mode is set to Automatic, the Tools will be automatically upgraded.
  • If the Tools upgrade mode is set to Manual, the system displays a message to ask the VM user whether to upgrade the Tools.

Set the Tools upgrade mode.

Both administrators and VM users can set the upgrade modes of Tools and intelligent network interface card (iNIC) drivers for VMs to fulfill user demands.

  • For an administrator: Set the Tools upgrade mode for each VM on the FusionManager. For details, see Changing VM Upgrade Mode.
  • For a VM user:
    • Windows VM: Click the Tools icon in the lower right corner on the VM desktop and set the upgrade mode.

      If the VM upgrade modes set by the administrator and the VM user are different, the VM upgrade mode in effect is Automatic.

    • Linux VM: The upgrade mode cannot be set on a Linux VM OS.
