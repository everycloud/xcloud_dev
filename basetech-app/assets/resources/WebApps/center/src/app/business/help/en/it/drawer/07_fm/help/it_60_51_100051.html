Hosts

Hosts are the physical computers running virtualization software and providing computing resources, such as CPUs and memory for VMs.

After a cluster is successfully associated, the hosts contained in the cluster are displayed on this page.

Host Power Supply Management

Host power supply management includes powering on, powering off, and safely restarting hosts.

The VMware hypervisor does not support the host power on function.

Setting a Host to Maintenance Mode

The host in maintenance mode is isolated from the entire system so that some operations, such as parts replacement or device maintenance, can be performed on the host without affecting services provided by the system.

If you perform operations, such as power-on and power-off, on the host that is in maintenance mode, its VMs become unavailable. To ensure VM availability, select Migrate All VMs on the Host.

In the VMware hypervisor, VMs on a host must be stopped or migrated to other hosts before the host is set to maintenance mode.

Click More and select Enter Maintenance Mode.

Antivirus Settings

You can enable the antivirus function for VMs running on a host only after the antivirus function is enabled for the host. However, the number of security user VMs must be less than the value configured in antivirus settings.

The VMware hypervisor does not support configuration of antivirus settings.

Click More and select Anti-Virus Setting.

Exporting Host Information

You can export host information to your local PC.
