Installing the Tools on a Linux VM

  • Scenarios
  • Prerequisites
  • Procedure
  • Additional Information

Scenarios

Tools provides drivers for VMs. It improves VM input/output (I/O) performance, monitors VM hardware, and enables other advanced functions on VMs. During VM template creation, install the Tools on the template.

For details about Tools functions, see Introduction.

Prerequisites

Conditions

  • The operating system (OS) has been installed on the VM.
  • Tools has not been installed on the VM.
  • The free space of the system disk is greater than 20 MB.
  • You have replaced the operating system kernel file for the VM if the VM runs the Red Hat Enterprise Linux 4.7/6.0 (32-/64-bit) or CentOS 4.7/6.0 (32-/64-bit). For details, see Replacing VM OS Kernel Files.
  • You have obtained the ISO file that contains the Tools installation file.

    The software package name is FusionCompute V100R005C00SPC200_GuestOSDrivers.zip, and the ISO file name is FusionCompute V100R005C00SPC200_GuestOS.iso.

    To obtain the software package
    • For enterprise users, visit http://support.huawei.com/enterprise and choose Software > IT > FusionCloud > FusionSphere > FusionCompute > V100R005C00SPC100.
    • For telecom carrier users, visit http://support.huawei.com and choose Software > Carrier Software & Core Network > Carrier Cloud Computing > IT Cloud Infrastructure Solution > Cloud OS > FusionCompute > V100R005C00SPC100.
    NOTE:
    Verify the obtained software packages. For details, see Verifying Software Packages.

Procedure

    Log in to the VM using VNC.

    1. Log in to the VM using Virtual Network Computing (VNC) as user root and mount a CD/DVD-ROM drive or ISO file to the VM.

      The FusionCompute V100R005C00SPC200_GuestOS.iso is the required ISO file to be mounted.

    Install the Tools.

    1. On the VM desktop displayed in the VNC window, right-click the desktop, and choose Open in Terminal.

      The command line interface (CLI) window is displayed.

    2. Run the following command to create the xvdd directory:

      mkdir xvdd

    3. Run the following commands to mount a CD/DVD-ROM drive to the VM:

      mount Path to be mounted xvdd

      For example, mount /dev/sr0 xvdd

      Table 1 lists the directories for mounting a CD/DVD-ROM drive or ISO file to a VM running the Linux OS.

    4. Run the following command to switch to the xvdd/pvDriver/linux directory:

      cd xvdd/pvDriver/linux

    5. Run the following command to view the installation package of the Tools:

      ls

      The following information is displayed: 
      ...
uvp-tools-linux-xxx.tar.bz2
uvp-tools-linux-xxx.tar.bz2.sha256

    6. Run the following commands to copy the Tools installation package to the root directory:

      cp uvp-tools-linux-xxx.tar.bz2 /root

      cd /root

      Table 2 lists the mapping between OS versions and Tools versions.

      Table 2 Mapping between OS versions and Tools versions

      OS Version

      Tools Version

      Tools Installation Command

      DOPRA ICTOM V002R003 EIMP

      pvdriver-dopra-eimp-64bit-xxx-xx.tar.bz2

      ./install -i

      DOPRA ICTOM V002R003 IMAOS

      pvdriver-dopra-ump-64bit-xxx-xx.tar.bz2

      Red Hat Enterprise Linux 3.0

      pvdriver-redhat3.0-32bit-xxx-xx.tar.bz2

      Red Hat Enterprise Linux 3.4

      pvdriver-redhat3.4-32bit-xxx-xx.tar.bz2

      Others

      uvp-tools-linux-xxx.tar.bz2

      ./install

    7. Run the following command to decompress the Tools:

      tar -xjvf uvp-tools-linux-xxx.tar.bz2

    8. Run the following command to switch to the directory in which the Tools installation package is stored:

      cd uvp-tools-linux-xxx

    9. Run the following command to install the Tools:

      ./install

      The Tools is installed if the following information is displayed: 
      The PV driver is installed successfully.
Reboot the system for the installation to take effect.

      NOTE:
      Run the ./install -i command to install the Tools if the VM runs one of the following OS:
      • DOPRA ICTOM V002R003 EIMP
      • DOPRA ICTOM V002R003 IMAOS
      • Red Hat Enterprise Linux 3.0
      • Red Hat Enterprise Linux 3.4

    Verify the installation.

    NOTE:

    After Tools is installed, restart the VM for Tools to take effect. If an iNIC driver is required, install the iNIC driver before performing the following operations to reduce VM restart times.

    1. Run the following command to restart the VM:

      reboot

    2. After the VM restarts, log in to the VM using VNC as user root.
    3. On the VM desktop displayed in the VNC window, right-click the desktop, and choose Open in Terminal.

      The command line interface (CLI) window is displayed.

    4. Run the following command to check the installation of the Tools:

      ps -ef | grep uvp-monitor

      root        4561      1   0  Jun29 ?          00:00:00  /usr/bin/uvp-monitor
root        4567   4561   0  Jun29 ?          00:00:00  /usr/bin/uvp-monitor
root        6185   6085   0  03:04  pts/2     00:00:00  grep uvp-monitor

      The Tools is installed successfully if the information in black is displayed as shown above.
