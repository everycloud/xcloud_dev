Application Templates

An application template is a service model that combines logical components, including VMs, software packages, scripts, and network resources, based on the configured relationship. You can use an application template to create VMs in batches and rapidly deploy application instances.

The system provides three application templates for commonly used services. You can also import an existing template to the system or create a custom template.

Obtaining an Application Template

You can obtain an application template in either of the following ways:

  1. Import an existing application template to FusionManager.

    Search for an application template from the Internet, such as from Forum, save it to your PC, and import the template to FusionManager.

  2. Customize an application template.

    Create a custom application template on FusionManager.

    For details, see Creating an Application Template.

  3. Copy an application template.

    Copy a published application template to create an identical application template and modify the template attributes as needed.

    Locate the row that contains the target application template, click More, and select Replicate.

Publishing an Application Instance

An application template customized in or imported to the FusionManager system can be used to provision application instances only after being published.

Locate the row that contains the target application template and click Publish.

Exporting an Application Template

Export a published application template to the local PC for data backup or statistical analysis.

Locate the row that contains the target application template, click More, and select Export.

Modifying an Application Template

You can modify the attributes, logical components, and configuration relationship of an application template. The information about a published application can be modified only after the template publishing is canceled.

Locate the row that contains the target application template, click More, and select Modify.

Deleting an Application Template

If an application template is not needed any more, you can delete it from the system. A published application can be deleted only after the template publishing is canceled.

Locate the row that contains the target application template, click More, and select Delete.
