IP Bandwidth Specifications

An IP bandwidth specification defines the maximum transmit and receive bandwidths for elastic IP addresses that can be used in a network.

You can use the specified IP bandwidth specifications when applying for elastic IP addresses for a network that uses hardware firewalls.

Modifying an IP Bandwidth Specification

You can modify all attributes of an existing IP bandwidth specification as needed. However, the specification attribute modification will not change the IP bandwidth attributes of the existing elastic IP addresses.

Deleting an IP Bandwidth Specification

If an IP bandwidth specification is not needed any more, you can delete it from the system.
