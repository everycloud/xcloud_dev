A user can have roles and permissions configured to meet its restricted system access requirements.

After FusionManager is installed, the system provides a default user account admin. You can use the admin account to log in to FusionManager the first time.

NOTE:
When creating or modifying a user, you can set a phone number and email address for the user as contact information. Ensure that you comply with local laws and regulation when using the specified personal information.

Privacy Statement:

  • Phone number: used for alarm notification by SMS. This function allows the system to send alarm information to the administrator or tenants using SMS messages after an alarm is generated. The system administrator can view the phone numbers specified for all VDC administrators and all tenants in the VDC. The administrator of a VDC can view the phone numbers specified for all tenants in the VDC. After a phone number is configured for a user, the number will be kept in the system until being deleted from the system. In this version of FusionManager, the alarm notification by SMS function has not yet been delivered for commercial use, though the phone number can be specified as part of the personal information of a user added to the system. However, the system will not use the collected phone number information for other purposes.
  • Email address: used for alarm notification by email. This function allows the system to send alarm information to the administrator or tenants using emails after an alarm is generated. The system administrator can view the email addresses specified for all VDC administrators and all tenants in the VDC. The administrator of a VDC can view the email addresses specified for all tenants in the VDC. After an email address is configured for a user, the email address will be kept in the system until being deleted from the system.

Creating a User

After logging into the system, you can create new users, assign different roles to the users, and configure the allowed system access duration and IP address for these uses based on their access requirements.

Choose User > User Management and click Create User.

Change your password after you log in to FusionManager for the first time and then periodically change the password to enhance account security.

Locking a User

To forbid a user from accessing the FusionManager system, you can lock the user.

Choose User > User Management, locate the row that contains the target user, and click Lock.

Unlocking a User

If the number of failed login attempts of a user reaches the upper limit, the user will be locked during the specified lockout period. You can unlock this user in the lockout period. The user lockout conditions can be specified in the password policy settings.

Choose User > User Management, locate the row that contains the target user, and click Unlock.

Modifying User Information

You can modify the role and rights settings of a user if needed.

Choose User > User Management, locate the row that contains the target user, click More, and select Modify.

Deleting a User

If a user account is not needed any more, you can delete it from the system. After a user is deleted, the resources allocated to the user are released.

Choose User > User Management, locate the row that contains the target user, click More, and select Delete.

The default user account admin cannot be deleted.

Setting a Password

If a user forgot the login password, you can log in to the system using any administrator account and reset the password for the user.

Choose User > User Management, locate the row that contains the target user, click More, and select Reset Password.
