Custom Reports

  • Creating a Custom Report
  • Modifying a Custom Report
  • Deleting a Custom Report
  • Downloading a Custom Report

Custom reports allow you to collect data on the required statistical objects and can be downloaded. If system reports cannot keep up with your requirements, you can create custom reports as required.

Creating a Custom Report

Custom reports can be used to collect statistics on resource objects, including the CPU and memory.

To create a custom report, choose Monitoring > Custom Reports and click Create.

Modifying a Custom Report

You can change the report name, statistics objects, and statistics indicators of an existing custom report.

To modify a custom report, choose Monitoring > System Reports, locate the row that contains the report, and click More > Modify.

Deleting a Custom Report

You can delete a custom report that is no longer used.

To delete a custom report, choose Monitoring > System Reports, locate the row that contains the report, and click More > Delete.

Downloading a Custom Report

You can download a custom report to your local PC to view its statistics.

To download a custom report, choose Monitoring > Custom Reports, locate the row that contains the report, and click Download Report.
