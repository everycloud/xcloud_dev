如何配置交换机组播功能

  • 操作场景
  • 前提条件
  • 操作步骤

操作场景

该任务指导系统管理员在使用VXLAN功能时，在交换机上配置组播功能，并开启三层组播转发。
说明：

以华为S5300 V100R005版本交换机设备为例，如下操作方法仅供参考，若相关的命令或参数发生变更，请以实际产品为准。其他型号设备的配置方法，请参考对应产品版本的产品文档。

在运行组播业务时，组播通道上需开启交换机的IGMP侦听功能和设备的查询器功能。一般可通过在交换机shell中执行相关命令开启，详细配置步骤请参见交换机的配置说明。

前提条件

  • 已使用终端仿真程序登录交换机设备。
  • 已在交换机上配置VTEP网络的VLAN信息，并且在汇聚交换机上配置对应VLAN的VLANif。VTEP网络包括FusionCompute上行链路的VTEP网络与FusionManager上添加的VTEP网络。

操作步骤

    开启交换机三层组播转发功能

    三层组播转发功能在汇聚交换机上进行配置。FusionCompute上行链路的VTEP网络与FusionManager上添加的VTEP网络都需要开启三层组播转发功能。

    1. 进入系统视图。

      <Quidway> system-view

    2. 打开组播三层转发的全局开关。

      [Quidway] multicast routing-enable

    3. 进入到DVS上行链路的VTEP网络的VLANif。

      [Quidway] interface Vlanif VLAN ID

      例如，进入到431的VLANif。

      [Quidway] interface Vlanif 431

    4. 对相应的VLANif打开三层组播转发功能。

      [Quidway-vlanif431] pim dm

      [Quidway-vlanif431] igmp enable

    5. 退出VLANif视图。

      [Quidway-vlanif431] q

    6. 对FusionManager中添加的VTEP网络的VLANif，打开三层组播转发功能，具体方法请参见3～5。

    开启交换机IGMP-SNOOPING功能

    IGMP-SNOOPING功能需要在汇聚交换机和接入交换机上进行配置。如果主机与汇聚交换机之间的链路上有多个接入交换机，这些交换机都需要开启IGMP-SNOOPING功能。

    1. 打开IGMP-SNOOPING全局开关。

      [Quidway] igmp-snooping enable

    2. 进入到DVS上行链路的VTEP网络的VLAN。

      [Quidway] vlan VLAN ID

      例如，FusionCompute中DVS上行链路的VTEP网络的VLAN ID为431。

      [Quidway] vlan 431

    3. 在相应的VLAN内启用IGMP-SNOOPING功能。

      [Quidway-vlan431] igmp-snooping enable

    4. 退出VLAN视图。

      [Quidway-vlan431] q

    5. 对FusionManager上添加的VTEP网络的VLAN，启用IGMP-SNOOPING功能，具体方法请参见8～10。
    6. 退出系统视图。

      [Quidway] q

    7. 退出交换机。

      <Quidway> q

      任务结束。
