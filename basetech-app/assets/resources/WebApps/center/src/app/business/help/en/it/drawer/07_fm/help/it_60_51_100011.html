Adaptation Package Management

Including More Compatible Devices in FusionManager

FusionManager can manage and monitor only the devices with their models specified in the FusionManager adaptation package.

The FusionManager system periodically updates its adaptation package. If you need to have more models of hardware devices compatible with FusionManager, download and install the latest adaptation package.

Viewing FusionManager-compatible Devices and Obtaining the Latest Adaptation Package

Log in to Compatibility check assistant to check the physical device models supported by FusionManager and determine whether to install the latest adaptation package based on the service requirements.

Installing the Adaptation Package

Perform the following steps to install the latest adaptation package to enable FusionManager to support more models of physical devices:
  1. Download the latest adaptation package and save it on your PC.
  2. Click Install Adaptation Package.

After the package is installed, the package information is displayed on the Adaptation Package page. The new hardware device models will be available for selection when you add physical devices or batch add templates to FusionManager.

Uninstalling the Adaptation Package

If the devices supported by an adaptation package are not required any more, you can uninstall the adaptation package. However, if some devices included in the adaptation package have been added to FusionManager, the adaptation package cannot be uninstalled.

Locate the row that contains the target adaptation package and click Uninstall.
