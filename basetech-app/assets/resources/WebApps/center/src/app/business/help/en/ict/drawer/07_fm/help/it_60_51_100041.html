Cloud Resource Pools

A cloud resource pool contains all resources managed in a FusionSphere OpenStack system.

FusionManager is able to manage multiple cloud resource pools.

Adding FusionSphere OpenStack to FusionManager

After a cloud resource pool is added to FusionManager, FusionManager can use and manage all the resources in the pool in a unified manner. A cloud resource pool can be connected to only one FusionManager system. Otherwise, system exceptions may occur.

Updating a Cloud Resource Pool

If VM or disk information in FusionSphere OpenStack has changed, update the information to .

Locate the row that contains the cloud resource pool, click More, and select Update.

Suspending or Resuming a Cloud Resource Pool

If the resources in a cloud resource pool are insufficient or you need to perform maintenance for the pool, for example, scale up the pool, suspend the cloud resource pool. In this case, however, using existing resources in the pool, such as VMs, is not affected.

Locate the row that contains the cloud resource pool, click More, and select the required operation.

Deleting a Cloud Resource Pool

If a cloud resource pool is not needed any more, for example, the pool is to be deleted for system capacity reduction, delete all resources in the resource pool and delete the cloud resource pool.

Locate the row that contains the cloud resource pool, and click Delete.
