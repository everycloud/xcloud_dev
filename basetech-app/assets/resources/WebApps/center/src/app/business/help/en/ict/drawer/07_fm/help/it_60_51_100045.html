VM-Hardware

A VM runs on a host and consists of virtual resources, including virtual CPUs, memory, disks, and network interface cards (NICs). Computing resources (such as CPUs and memory) and network connections are provided by hosts; virtual disk storage space is provided physical storage devices.

Modify hardware attributes and configurations on demand after the VM is created.

Modifying Specifications

Adjust VM performance by modifying VM specifications, including the number of CPUs and the memory size.

For details about requirements, see VM Migration and Specification Modification Requirements.

The specifications of a VM can be modified only when the VM is in the Stopped or Running state. If a VM is running, it will be automatically stopped before its specifications are modified. After the VM specifications are modified, the VM automatically returns to the Running state.

After clicking Modify, you also need to click Confirm Modification to make the system continue proceeding with the subsequent steps.
