Adding Mode

FusionManager supports two modes for adding physical devices:
  • Single-device add: You can add each single device one by one to FusionManager. This mode applies to tasks for maintaining devices individually. To add devices one by one, choose Resource > Devices on the FusionManager web client, select your desired device type, and create a device of that type on the displayed page.
  • Batch add: Import the physical device information table so that all devices listed in the table can be added to FusionManager at a time. This mode applies to tasks for maintaining devices in batches in the initial system configuration phase. To add devices in batches, choose Resource > Devices > Batch Add Physical Devices on the FusionManager web client and perform steps as instructed.

Procedure for Adding Physical Devices to FusionManager in Batches

  1. Click Download Template to download the device information template file and save the file to your PC.
  2. Fill the template with the required information and save the settings.
    NOTE:

    The template does not support setting SNMPv3 as the alarm reporting protocol for the connected devices. If you require the connected devices to use SNMPv3, add each target device to FusionManager one by one on the FusionManager Web Client.

  3. Import the configured template file to FusionManager.
  4. After the physical devices are successfully added to the system, keep the template file properly or destroy the template file to prevent security risks resulted from the leak of device information contained in the template file.
