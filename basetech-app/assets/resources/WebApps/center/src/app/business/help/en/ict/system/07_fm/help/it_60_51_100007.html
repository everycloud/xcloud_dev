SAN Devices

  • Adding a SAN Device to FusionManager
  • Deleting a SAN Device

Storage Area Network (SAN) is a storage architecture that connects the storage system to the application server over the network. SAN storage devices provide data storage space for VMs in the system using shared software to provide block-level data sharing service.

Add SAN devices to FusionManager so that FusionManager can monitor and manage these devices in a unified manner.

Adding a SAN Device to FusionManager

Choose Resource > Devices > Storage Devices > SAN and click Add SAN Device.

After a SAN device is added to FusionManager, the SAN device information is displayed on the SAN page.

If the required server model is not available for selection, include more compatible hardware devices into the system. For details, see Adaptation Package Management.

Deleting a SAN Device

If a SAN device is not needed any more, you can delete it from the system. After a SAN device is deleted, FusionManager will not manage or monitor it, but the deletion will not affect the virtual resources assigned to the resource pool the deleted SAN device originally belongs to.

Choose Resource > Devices > Storage Devices > SAN, locate the row that contains the target SAN device, and click Delete.
