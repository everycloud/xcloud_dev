在Windows系统中安装智能网卡驱动

  • 操作场景

操作场景

当主机使用智能网卡时，运行在该主机上的虚拟机必须安装智能网卡驱动，否则虚拟机的网卡不可见，导致虚拟机网络不通。

当有一台主机使用了智能网卡时，建议所有虚拟机都安装智能网卡驱动，避免虚拟机迁移至使用智能网卡的主机时，出现网络故障。

部分虚拟机操作系统不支持iNIC智能网卡，具体信息请参见操作系统兼容性列表。不建议将使用这些操作系统的虚拟机部署在使用智能网卡的主机上。

必备事项

前提条件

  • 已登录FusionManager。

  • 虚拟机的状态为“运行中”。

  • 虚拟机已安装Tools，且Tools运行正常。

  • 已获取智能网卡驱动所在的ISO文件。

    软件包名称为“FusionCompute V100R005C00SPC200_GuestOSDrivers.zip”，解压缩后的ISO文件名称为“FusionCompute V100R005C00SPC200_GuestOS.iso”。

    软件包下载路径：

    • 运营商用户：“http://support.huawei.com > 产品软件 > 电信软件与核心网 > 电信云计算 > IT云基础设施解决方案 > Cloud OS > FusionCompute > FusionCompute V100R005 > FusionCompute V100R005C00SPC100”
    说明：
    获取到软件包后建议先执行软件包校验的操作，具体操作请参见校验软件包。

操作步骤

    判断是否需要安装智能网卡

    1. 在FusionManager界面，在“资源 > 设备 > 服务器”中选择主机。
    2. 在“硬件”页面，单击“网卡”。

      查看网卡的“型号”。

      根据型号辨识网卡是否为智能网卡。

    安装智能网卡驱动

    1. 使用VNC方式登录虚拟机，并挂载光驱。

      挂载光驱时需挂载“FusionCompute V100R005C00SPC200_GuestOS.iso”。

    2. 在VNC登录窗口的虚拟机操作系统界面，选择“开始 > 计算机”，双击虚拟光驱图标。

      打开存放智能网卡驱动安装程序的目录。

    3. 在“iNICDriver/windows”文件夹内，双击智能网卡驱动的安装程序，根据页面提示完成智能网卡驱动的安装。

      操作系统和智能网卡驱动的对应关系如表1所示。

      表1 操作系统和智能网卡驱动的对应关系

      操作系统

      智能网卡驱动

      Windows 2000

      iNIC_Installer_Win2000_x.x.x.xx.exe

      Windows XP

      iNIC_Installer_XP_x.x.x.xx.exe

      Windows Server 2003

      iNIC_Installer_Win2003_x.x.x.xx.exe

      Windows Server 2008

      iNIC_Installer_WinVista2008_x.x.x.xx.exe

      Windows Server 2008 R2

      iNIC_Installer_Win2008r2_x.x.x.xx.exe

      Windows 7

      iNIC_Installer_Win7_x.x.x.xx.exe

      Windows 8

      iNIC_Installer_Win8_x.x.x.xx.exe

    4. 卸载光驱。
