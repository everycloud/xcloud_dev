MAC Address Resource Pool

In a MAC address resource pool, you can specify one or multiple MAC address segments available for FusionManager to assign MAC addresses to VM network interface cards (NICs).

Adding a MAC Address Resource Pool

Create an empty MAC address resource pool for adding MAC address segments to it.

Deleting a MAC Resource Pool

Delete an empty MAC address resource pool as needed.

Adding a MAC Address Segment

Add a MAC address segment in a MAC address resource pool by configuring the start and end MAC addresses in the segment.

Deleting a MAC Address Segment

Delete a MAC address segment from a MAC address resource pool. The deleted MAC address segment can be added back to the pool if needed.
