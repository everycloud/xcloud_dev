VLBs

The VLBs page displays all virtual load balancers (VLBs) created in the virtual private clouds (VPCs) of the resource zone. A VLB provides load balancing services.

VLBs can be classified into the following types:
  • Hardware VLBs: provided by a physical load balancer (LB).
  • Software VLBs: provided by a system service VMs in the FusionManager.

Obtaining a VLB

Prerequisites

  • To obtain a hardware VLB, ensure that you have added the required physical LB to FusionManager.
  • To obtain a software VLB, ensure that you have added a Virtual Software Appliance (VSA) management network in the network resource pool of the current resource zone and have imported the VSA VM template to the system.
