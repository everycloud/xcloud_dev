VPN Gateways

  • Creating a VPN Gateway
  • Deleting a VPN Gateway

The VPN Gateways page displays all the virtual private network (VPN) gateways created in the current cloud resource pool.

A virtual private network (VPN) provides an encrypted communication channel between remote users and their virtual private clouds (VPCs) so that the users can directly obtain VPC service resources through the VPN. The VPN gateway provides routing functions for VPN channels on a VPC.

Creating a VPN Gateway

Create a VPN gateway for a shared virtual private cloud (VPC). The system uses an elastic IP address as the VPN gateway.

Choose VDC > VPC > VPN > VPN Gateways and click Create VPN Gateway.

Deleting a VPN Gateway

If a VPN gateway in a shared VPC is not needed any more, you can delete it from the system. However, if the VPN gateway is in use by a VPN connection, the gateway cannot be deleted.

Choose VDC > VPC > VPN > VPN Gateways, locate the row that contains the target VPN gateway, and click Delete.
