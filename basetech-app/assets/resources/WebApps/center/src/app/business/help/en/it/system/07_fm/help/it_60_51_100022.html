DVSs

  • Creating a DVS Using VSS

Functioning the same as a physical switch does, a distributed virtual switch (DVS) supports data switching among VMs and between VMs and external networks.

A DVS is not required to be independently created in the system. This is because the DVSs in a resource cluster can be included in a FusionManager resource zone when the resource cluster is associated with the resource zone on the Computing Resource Pools page. The included DVSs are displayed on the DVSs page after association.

Creating a DVS Using VSS

If the VMware Virtual Software Switch (VSS) is used, you need to convert the VSS to a FusionManager-supported DVS first.

Choose Resource > Resource Zones, select a resource zone, choose Network Resource Pools > DVS, and click Create.
