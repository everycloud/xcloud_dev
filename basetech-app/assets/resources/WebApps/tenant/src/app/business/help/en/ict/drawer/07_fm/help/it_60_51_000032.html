Provisioning an Application Instance

Description

An application instance is a collection of VMs and service software that provides specific services, for example, SQL Server, customer relationship management (CRM), and enterprise resource planning (ERP).

Provisioning an application instance is a process of creating an application instance.

Provisioning Method

You can provision application instances using a method shown in Figure 1. Table 1 lists the involved tasks.

Web Client Navigation Path

Create an application template.

An application template is a service model that combines logical components, including VMs, software packages, scripts, and network resources, based on the configured relationship.

You can create an application template or use an application template created by the administrator. If you use an existing application template, skip this step.

 Choose Application > Application Templates.

Create an application instance.

Use the desired application template to create an application instance.

 Choose Application > Application Instance.
