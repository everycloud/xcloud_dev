Login and Logout

Scenarios

Log in to ServiceCenter and perform operations on it. After operations are complete, log out of ServiceCenter to ensure user security.

Prerequisites

  • You have installed the browser of the required version. If you use Internet Explorer or Mozilla Firefox, ensure that the browser you use has been configured properly.
  • You have obtained the floating IP address of the ServiceCenter node.
  • You have obtained the username, password, and user domain for logging in to ServiceCenter.
  • The required resolution has been set on the local PC.

Login Using Internet Explorer

The following uses Internet Explorer 9.0 as an example to describe how to log in to ServiceCenter.
  1. Open Internet Explorer.
  2. Enter the network address of ServiceCenter in the address box and press Enter.

    The network address format is http://ServiceCenter floating IP address.

    For example, enter http://192.168.40.2.

  3. If the security certificate is not installed, a security warning dialog box is displayed. Click Continue to this website (not recommended) to continue login.

    If the Firefox or Google Chrome browser is used, proceed as required:

    • If the browser is Firefox, click Add Exception.
    • If the browser is Google Chrome, click Advanced.

  4. Enter Username and Password.
  5. Select a view type.
  6. Enter the verification code.
  7. Click Login.

    The ServiceCenter page is displayed.

    If the system displays a message indicating that the login password must be changed upon the first login or be reset, change the password as instructed.

    NOTE:

    On the ServiceCenter Web Client, do not press Backspace after a dialog box is displayed. Otherwise, the Web Client may fail to work properly. If this situation occurs, you can refresh the page.

Logout

In the upper right corner of the login page, click username and select Logout to log out of ServiceCenter.

FAQ

How Do I Log in to two ServiceCenter Web Clients on the Same PC?
To log in to two ServiceCenter Web Clients on your local PC, ensure that the following requirements are met:
  • Different usernames are used for logging in to two ServiceCenter Web Clients.
  • The ServiceCenter Web Clients are opened in different browsers or on different pages in one browser.
