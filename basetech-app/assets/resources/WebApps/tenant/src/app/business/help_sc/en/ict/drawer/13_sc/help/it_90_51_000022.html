Applying for a Dedicated Server

Overview

Apply for a dedicated server instance of certain specifications so that you can configure the dedicated server and use it to provide a service.

Prerequisites

  • A dedicated server has been added to Cloud Resource Pool on ServiceCenter.
  • A dedicated server service that meets requirements exists in the service list. If such a dedicated server service does not exist, the system administrator needs to create the service first.
