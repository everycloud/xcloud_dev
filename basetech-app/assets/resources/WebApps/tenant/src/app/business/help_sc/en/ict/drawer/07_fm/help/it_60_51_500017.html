Tasks and Logs

Task Center

The task center displays the results of some tasks that cannot immediately produce a result or take a long time to finish. You can view the task execution details in real time.

To switch to the task center, choose Monitoring > Task Center.

Operation Logs

You can view operation logs to learn detailed information about an user operation, such as the specific action and the operation time. If the content of an operation log shows system as the operator, the operation is performed by the system, rather than the user who is viewing the operation log.
