Domain Management

A domain is used to classify VMs in a virtual data center (VDC).

To allow different users to have different operation permissions for various VMs in a domain, you can add these users to the domain and assign different permissions to them.

Creating a Domain

If the default domain in the system cannot meet the requirements for fine-grained user rights management, you can create a new domain.

Choose User > Domain Management and click Add.

Deleting a Domain

If a domain is not needed any more, you can delete it from the system.

Choose User > Domain Management, select the target domain, and click Delete.

Adding a Domain User

To allow a user to have rights granted to a domain, you can add the user to the domain.

Choose User > Domain Management, navigate to the Users page in the Details area on the right pane, and click Add Member.

Removing a Domain Member

To disqualify a domain member from using rights granted to the domain, you can remove the user from the domain.

Choose User > Domain Management, navigate to the Users page in the Details area on the right pane, select the target user, and click Remove.

Modifying Rights of a User

You can modify the operation permissions of a user if required.

Choose User > Domain Management, navigate to the Users page in the Details area on the right pane, select the target user, and click Modify Rights.

Adding VMs to a Domain

To have some VMs be managed within a domain, you can add these VMs to the domain.

Choose User > Domain Management, navigate to the Details box on the right side of the page, click the VM tab, click Batch, and select the VMs to be added to the domain.

Deleting a VM from a Domain

If a VM in a domain does not need to be managed within the domain, you can delete this VM from the domain.

Choose User > Domain Management, navigate to the Details box on the right side of the page, click the VM tab, select the VM to be deleted from the domain, and click Delete.
