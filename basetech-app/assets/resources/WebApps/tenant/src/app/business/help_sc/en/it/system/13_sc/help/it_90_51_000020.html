Introduction

A cloud hard disk provides block storage device services with persistence and high availability.

Applying for a cloud hard disk indicates the process for applying for and creating a cloud hard disk.

Prerequisites

Cloud hard disk service that meet the application requirements already exist in the service list. If not, the administrator must create related services first.
