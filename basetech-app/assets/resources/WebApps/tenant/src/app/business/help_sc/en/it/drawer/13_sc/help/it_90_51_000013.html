My Applications

My Applications displays all applications submitted by users. It is used for users to query the status of service approval and provisioning. Only resources that do not expire and are in the Succeeded status can be used by users.

Modifying an Application

When a service application is rejected by the administrator, users can modify the application and submit it again.

When an application is rejected by the administrator, the status of this application is turned to To Submit. Users can modify the application order.

Click Modify in Operation from the row where the application order locates.

Revoking an Application

When a service application is rejected by the administrator, users can close the application that is submitted by revoking it. An application cannot be modified and submitted again after it is revoked. If you want to apply for the service, you need to submit a new application.

When an application is revoked, the status of this application is turned to Revoke.

Click Revoke in Operation from the row where the application order locates.
