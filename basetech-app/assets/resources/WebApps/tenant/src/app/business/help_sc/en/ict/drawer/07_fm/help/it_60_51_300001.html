Application Instances

An application instance is a collection of VMs and service software that provides specific services, for example, SQL Server, customer relationship management (CRM), and enterprise resource planning (ERP).

Application Instance Power Management

Power management for an application instance includes start and stop the application instance. Essentially, stopping an application instance stops all VMs included in the application instance, and start the application instance starts all VMs in the application instance.

Locate the row that contains the target application instance and select the desired operation to perform.
