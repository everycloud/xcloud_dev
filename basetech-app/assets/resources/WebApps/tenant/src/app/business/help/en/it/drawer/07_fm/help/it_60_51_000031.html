Provisioning a VM

Overview

A virtual machine (VM) is a software computer that functions like a physical computer, running an operating system (OS) and applications.

Provisioning a VM is a process of creating a VM with specified configurations.

Provisioning Method

You can provision VMs using a method shown in Figure 1 based on the mapping scenario listed in Table 1. Table 2 lists the involved tasks.

Figure 1 Provisioning method

Table 1 Scenarios

Provisioning Mode

Scenario

1

The VM template created by the system administrator meets the requirements of the VM to be provided. In this case, use the created VM template to create the target VM.

2

Multiple VMs need to be provisioned at a time and managed in one application instance. In this case, create an application template using a VM template and create VMs in batches using the application template.

3
The VM template created by the system administrator cannot meet the requirements of the VM to be provisioned. In this case, you can use one of the following methods:
  • Request the system administrator to create a VM template based on the required specifications.
  • Create a VM (parent VM) using an existing VM template, adjust the parent VM specifications to the required specifications, convert the adjusted parent VM into a new VM template, and use this new VM template to create the required VM.
