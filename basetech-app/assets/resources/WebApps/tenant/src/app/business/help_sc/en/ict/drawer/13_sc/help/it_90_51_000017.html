Applying for a Cloud Host (Virtual Machine)

Introduction

Applying for a VM indicates the process that users apply for VM instances based on specific specifications and requirements for the VM to run applications and provide services.

Prerequisites

  • VM templates required by users are created on FusionManager.
  • Cloud host service that meet the application requirements already exist in the service list. If not, the administrator must create related services first.
